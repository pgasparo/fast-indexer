# Fast Indexing Code

This project aims to speed up indexers used by the JungfrauMX project.

1. xgandalf (https://gitlab.desy.de/thomas.white/xgandalf/-/tree/main)

# Source structure

* compile-xgandalf.txt: How to compile xgandalf
* xgandalf: xgandalf source
* xgandalf-build: xgandalf build
* xgandalf-install: xgandalf installation
* test: Test source and data
* test-build: Test build and installation
* test.sh: Call test executable
* xgandalf-snaphot.txt: git tag for last used xgandalf source
